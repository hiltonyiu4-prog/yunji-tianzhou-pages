总判断

晋文不是单一公司类型，而是以项目为入口、以平台为中枢、以生态为结果的企业家服务组织

业务

晋文到底做什么

晋文做云南本地项目，也链接全球企业家资源

三层业务结构

项目层
解决云南本地项目怎么定位、怎么招商、怎么入驻、怎么经营、怎么形成现金流
平台层
解决企业家平台怎么搭起来、客户怎么沉淀、资源怎么组织起来、活动和内容怎么持续起来
生态层
解决跨项目、跨圈层、跨区域、跨国际的协同，最终形成区域生态组织
很重要的特征
  • 本地端 — 扎根云南，做真实项目
  • 全球端 — 链接全球企业家资源，把云南项目放到更大的资源网络里去放大

晋文主要做六件事

  1. 帮云南本地项目找方向
    包括项目定位、业态组合、招商逻辑、客户判断、对外表达统一。不是简单"接项目"，而是先把云南本地项目的方向、逻辑和价值讲清楚，同时预留全球企业家资源导入和外部链接的接口。
  2. 帮项目把资源找来、把客户招进来
    包括企业家资源、商协会资源、政企资源、品牌资源、渠道资源和合作资源，也包括全球企业家资源导入。晋文做的不是空泛连接，而是把真实资源带进真实项目，把云南项目接到更大的全球资源网络中。
  3. 帮项目把招商做成交
    包括招商方案、联合创始人招募、入驻洽谈、权益设计、签约推进和落地执行。招商不是口号，而是晋文第一阶段最现实的现金流入口，也是把云南项目接入更大企业家网络的起点。
  4. 帮项目把企业家平台搭起来
    包括联合创始人体系、会员体系、资源伙伴体系、活动体系和内容体系。晋文不是只做单点招商，而是要把项目升级成能够沉淀客户、资源和关系的平台，并为后续链接全球企业家资源预留平台能力。
  5. 帮项目把订单和渠道接进来
    包括采购导入、圈层转介绍、内部订单、直播电商、选品、会务和活动资源导流，也包括面向更广企业家网络的渠道导入。晋文不只是招商，还要让项目有持续的业务流和订单流，并具备向外链接、向外放大的能力。
  6. 帮项目沉淀成长期平台和区域生态
    把一个个项目逐步做成企业家平台、资源平台和区域生态平台。平台不是一开始就有，而是从项目中长出来；生态也不是口号，而是从平台沉淀出来的。形成"云南主场 + 全球链接"的平台格局。

晋文的业务边界

晋文做什么，不做什么

✓ 晋文明确要做什么
  1. 项目战略定位
  2. 文化赋能与品牌表达
  3. 资源组织与撮合

  1. 不做泛化型招商代理

  2. 不做重执行外包公司
    不把自己做成全链条施工总包、物业代管
  3. 不做纯培训机构
    可以做活动、沙龙、访谈，但不以课程售卖为核心
  4. 不做空转平台
    可以有线上平台和AI工具，但不能过早重技术化包装
  5. 不做失控的关系撮合
    资源链接必须有规则、边界、记录、责任和复盘
  6. 不做单纯资本叙事
    可以有股权和联合创始人设计，但当前主线仍是项目落地
边界意义
边界不是为了限制，而是为了确保晋文不发散、不断线、不空转。

愿意长期、高投入、深合作进入体系的人

联合创始人席位、战略合作机制、资源入股与联营机制

能与晋文形成长期共建的合作方

特征

最独一的特征

区域性 + 深耕性 + 全球资源

1. 区域性：云南不是市场，云南是主场

晋文不是外部平台来做云南项目，而是以云南本地的产业肌理、资源结构、合作习惯和人情网络为基础，真正把云南当作自己的主场来经营。

2. 深耕性：创始人几十年深耕云南

这不是一段履历，而是能力底盘。它带来的不是单次资源，而是长期建立的信任网络、地方规则熟悉度、产业判断力、项目推进节奏感和对真实需求的理解。

3. 全球资源：云南主场之上，叠加全球企业家网络

晋文拥有上万全球企业家资源，可以把云南项目放到更大的企业家网络里组织和放大，让云南成为资源汇聚点，而不是资源外流点。

4. 独一组合：本地深耕力 + 全球链接力同时存在

这三者叠加后，晋文不是单纯的云南公司，也不是单纯的全国平台，而是一个以云南为主场、以深耕为底盘、以全球企业家资源为放大器的项目型生态组织。
